# rncli

### install

```bash
npm install @rayhomie/rncli -g
```
### check

```bash 
# View version
rncli -V
# or
rncli --version

# for hlep
rncli --help
```
### use

Open the terminal and enter the following to create the project
```bash
rncli create your_project_name
```

### start

```bash
cd ./your_project_name
yarn ios 
# or
yarn android
```
